# OpenShift Huawei Cloud MaaS Skill

This child skill package focuses on connecting browser-based coding workspaces, especially OpenShift Dev Spaces or Eclipse Che, with Cline and Huawei Cloud MaaS through an OpenAI-compatible endpoint. It is intended for AI coding scenarios where developers use hosted VS Code environments instead of local IDEs.

## Included Assets

- [SKILL.md](./SKILL.md): Main skill definition, workflow, and troubleshooting guidance
- [references/](./references): Integration notes for Dev Spaces, Cline, and MaaS field mapping
- [scripts/](./scripts): Verification and configuration helper scripts
- [agents/](./agents): Agent configuration used by the skill package

## Typical Use

- Configure Cline for Huawei Cloud MaaS
- Validate OpenAI-compatible endpoint behavior
- Troubleshoot browser-based AI coding workspace issues
- Separate platform, editor, and MaaS compatibility problems
